estrutura msg = : 03 05 01 FF00 F8

Lendo   saida   digital: 0X01
Lendo entrada   digital: 0X02
Escreve saida   digital: 0X05
Lendo entrada analogica: 0X06
Escreve saida analogica: 0X07

pinos saidas digitais: 
OK * IO2(SENTIDO HORARIO 		- FECHA JANELA SALA DE JANTAR)
OK * IO3(SENTIDO ANTI-HORARIO 	- ABRE  JANELA SALA DE JANTAR)
OK * IO4(SENTIDO HORARIO 		- FECHA JANELA SALA DE  ESTAR)
OK * IO5(SENTIDO ANTI-HORARIO 	- ABRE  JANELA SALA DE  ESTAR)
OK * IO6 e IO7(Enable nos motores). //por enquanto padrao HIGH e LOW, 
mas pode ser usado como pwm para ter o controle da velocidade dos motores.
OK * IO8(solenoide tranca)
OK * IO9(aquecedor)
OK * IO10(sirene) 

pinos entradas digitais: 
OK * IO11(sensor DS18B20)
OK * IO12(sensor   porta)

pinos saidas analogica: 
OK *AD4 e AD5(atuadores para lampadas da sala estar e quarto) dimerizadas, 
sendo 0V para desligado e 5V como brilho max
OK *AD6(atuador de ajuste do AC) // 1,7V igual a 17 graus celsius, 2,5V 25 graus celsius...

pinos entradas analogica: 
OK * AD0(Anemômetro) // 0,5V é equivalente a 5 km/h e 4,5V é 150 km/h.
OK * AD1(LM35) // Quando temperatura inferior a 17 graus liga aquecedor 
OK * AD2 e AD3(sensores cortina sala jantar e estar) // sendo 1V para 
totalmente fechado e 4V totalmente aberto. fazer offset para 0 a 3
